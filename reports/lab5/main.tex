\subsection{Метод Immersed Solid}

Immersed Solid~--- метод моделювання руху твердих тіл усередині рідини без необхідності створення рухомої сітки (moving mesh) або ковзних інтерфейсів (sliding interfaces).

Принцип дії: тверде тіло (мішалка) <<занурюється>> в загальну сітку рідинного домену. Елементи сітки, що перекриваються з твердим тілом, отримують додаткові силові члени (source terms), які імітують присутність твердого тіла. Швидкість рідини в зоні твердого тіла примусово задається рівною швидкості тіла (обертання).

Переваги: не потрібна окрема сітка для обертової зони; простіша побудова моделі; можливість моделювання складних рухів.

Недоліки: менша точність біля поверхні твердого тіла порівняно з методом ковзної сітки (Sliding Mesh).

\subsection{Параметри перемішування}

Часовий крок (Timestep):
\begin{equation}
  \Delta t = \frac{60}{\mathrm{RPM} \times N_{\mathrm{steps/rev}}},
\end{equation}
де $\mathrm{RPM}$~--- швидкість обертання, $N_{\mathrm{steps/rev}}$~--- кількість кроків за один оберт (рекомендовано 36--72).

Для варіанту~13 ($\mathrm{RPM} = 400$, $N_{\mathrm{steps/rev}} = 36$):
\[
  \Delta t = \frac{60}{400 \times 36} \approx 4{,}17 \times 10^{-3}~\text{с}.
\]

Число Рейнольдса для перемішування:
\begin{equation}
  \mathrm{Re_{mix}} = \frac{\rho\, N\, D^2}{\mu},
\end{equation}
де $N$~--- швидкість обертання (об/с), $D$~--- діаметр мішалки. При $\mathrm{Re_{mix}} > 10000$~--- турбулентний режим.

Коефіцієнт потужності (Power Number):
\begin{equation}
  N_p = \frac{P}{\rho\, N^3\, D^5},
\end{equation}
де $P$~--- потужність, що споживається мішалкою.

Потужність:
\begin{equation}
  P = M\,\omega,
\end{equation}
де $M$~--- крутний момент (Torque), $\omega$~--- кутова швидкість.

\section{Хронологічна послідовність виконання}

\begin{enumerate}
  \item \textbf{Створення геометрії.}
    Використано внутрішній об'єм ємності з Практичної роботи~№1. Додано якірну мішалку (Anchor impeller), розташовану по центру ємності. Мішалка змодельована як тверде тіло.

  \item \textbf{Побудова сітки (Meshing).}
    Побудовано об'ємну сітку для рідинного домену з достатньою деталізацією в зоні роботи мішалки.

  \item \textbf{Налаштування фізичної моделі (CFX-Pre).}
    \begin{itemize}
      \item Створено Fluid Domain: Water;
      \item Активовано підхід Immersed Solid для мішалки;
      \item Задано Angular Velocity: 400~об/хв ($\approx 41{,}9$~рад/с);
      \item Вісь обертання: $Y$ (вертикальна);
      \item Analysis Type: Transient;
      \item Total Time: 4~с;
      \item Timestep: $\approx 4{,}17 \times 10^{-3}$~с.
    \end{itemize}

  \item \textbf{Граничні умови.}
    \begin{itemize}
      \item Стінки ємності: Wall (No Slip);
      \item Верхня поверхня: Free Surface / Symmetry;
      \item Дно: Wall (No Slip).
    \end{itemize}

  \item \textbf{Розрахунок.}
    Запущено Transient розрахунок.

  \item \textbf{Анімація.}
    Створено анімацію (.mp4) процесу перемішування у реальному часі.
\end{enumerate}
