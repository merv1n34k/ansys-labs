\subsection{Ключові формули}

Еквівалентні напруження за теорією Мізеса:
\begin{equation}
  \sigma_{\mathrm{eq}} = \sqrt{\frac{1}{2}\left[
    (\sigma_1 - \sigma_2)^2 +
    (\sigma_2 - \sigma_3)^2 +
    (\sigma_3 - \sigma_1)^2
  \right]}.
\end{equation}

Повне переміщення:
$U_{\mathrm{total}} = \sqrt{U_x^2 + U_y^2 + U_z^2}$.
Запас міцності:
$n = \sigma_{\mathrm{yield}} / \sigma_{\mathrm{eq}}$;
$n \geq 1$ --- конструкція витримує навантаження.

\section{Аналіз та висновок}
% ===================================================================

Для оцінки міцності порівнюємо максимальне еквівалентне напруження з межею текучості:
\[
  \sigma_{\mathrm{eq,max}} = \result{X{,}XX}~\text{МПа}
  \quad\text{vs}\quad
  \sigma_{\mathrm{yield}} = 207~\text{МПа}.
\]

Запас міцності:
\[
  n = \frac{\sigma_{\mathrm{yield}}}{\sigma_{\mathrm{eq,max}}}
    = \frac{207}{\result{X{,}XX}}
    = \result{X{,}XX}.
\]
